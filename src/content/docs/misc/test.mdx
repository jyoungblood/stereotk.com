Code block with a filename and highlights:

export const exampleCode = `console.log('This could come from a file or CMS!');`
export const fileName = 'example.js'
export const highlights = ['file', 'CMS']

<Code code={exampleCode} lang="js" title={fileName} mark={highlights} />

---

Code blocks with tabs and different languages:

<Tabs>
  <TabItem label="Blade">
    <Code code="<h1>Hello world! My name is {{ $name }}</h1>" lang="blade" />
  </TabItem>
  <TabItem label="PHP">
    <Code code="<?php echo 'Hello world!'; ?>" lang="php" />
  </TabItem>
  <TabItem label="JS">
    <Code code="console.log('Hello world!');" lang="js" />
  </TabItem>
  <TabItem label="CSS">
    <Code code="body { color: red; }" lang="css" />
  </TabItem>
</Tabs>
